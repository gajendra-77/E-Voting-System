# 🗳️ E-Voting System

## About

This is a simple E-Voting System project.
Users can login, vote, and see results online.
This project is divided into frontend (UI) and backend (logic).

---

## Features

* Login system
* Vote casting
* Results display
* No duplicate voting

---

## Technologies

* Python
* HTML, CSS

---

## Run

```bash
python app.py
```
